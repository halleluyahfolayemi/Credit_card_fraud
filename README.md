# Data Analysis Project 

## Overview
This is a completed data analysis capstone project focused on extracting insights from real-world datasets using data analytics tools.

## Problem statement
SecureBank Nigeria has noticed unusual patterns in credit card transactions and wants to analyze 
spending behaviors, identify transaction anomalies, and understand customer usage patterns better. 
This analysis will help improve fraud detection protocols and enhance customer service by 
understanding normal transaction behaviors.

## Objectives
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Insight generation
- Data visualization
- Reporting findings

## Tools to be used
- Excel
- Power BI and SQL
- Microsoft Word and PowerPoint
